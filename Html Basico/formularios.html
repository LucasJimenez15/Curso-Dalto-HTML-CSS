Veremos lo que son los formularios, estos se basan en la solicitud y respuesta con el servidor y el cliente, sirven para solicitarle o pedirle informacion al usuario, es decir que estos nos den informacion sobre su contacto, telefono, nombre, etc. veremos los tipos de input es decir la casillita de marcado, o para agregar texto etc.

Para ello se utiliza la etiqueta form, esta es una etiqueta de bloque que es donde se contendra el formulario, y luego dentro de ella utilizaremos etiquetas input, en este caso etiquetas con el tipo input type="y elegiremos el tipo de input"

Hay 4 atributos importantes en los input estos son:

  • required es un atriubto de funcionalidad este lo que hace es obligar al usuario a escribir algo, por ejemplo si ponemos input type="text" required , esto nos pedira obligatoriamente que escribamos algo
  • name="" este es como un idetificado de los inputs, es de configuracion interna para el desarrollador, le sirve al servidor para saber de que campo estamos hablando es como un identificador para cuando le enviemos info al servidor ya sepa de que se trata, por ejemplo input type="text" name="username" , entoces el servidor ya sabra que se debe verificar el campo USERNAME
  • este es un atributo visual y es el placeholder="" es mostrarnos un pequeño texto cuando el usuario empieza a escribir
  • value esto es lo que va a recibir el servidor, es decir lo que nosotros pongamos en el input se almacenara en este atributo, por ejemplo tenemos input type="text" value="asdasd" lo que nosotros pongamos en el input o fomulario nos lo cargara en donde dice value="asdasd"
  • tambien tenemos el minlenght="1,2,3,etc..." que es basicamente un rango minimo para poner en un campo de tipo texto, esto depende mucho del caso
